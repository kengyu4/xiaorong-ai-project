# Agent 工作规则

## 目的

本 workspace 是「小绒老师助教」项目（AI 刷题学习应用）。所有工作围绕 dev-docs/ 中的设计文档展开，保持文档与代码一致。

## 约束

1. **先读索引再编码**：每次任务前必须读 `dev-docs/README.md`，定位相关文档，确认改动不与既有设计冲突。
2. **改动必更文档**：任何代码改动须同步更新 `dev-docs/` 中对应文档（API、DB、组件、部署等），索引不准时更新 `README.md`。
3. **新增文档必入索引**：新文件放 `dev-docs/`，并在 `README.md` 索引中登记。
4. **三条红线**：不读索引直接编码、改动不更新文档、新增文档不更新索引 —— 触任何一条即违规。
